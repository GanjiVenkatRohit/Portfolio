My Projects

IPL Match Win Prediction using ML

This project involved developing a robust IPL win prediction system using Python. I performed extensive data analysis with Pandas and NumPy, extracting over 10 distinct features and visualizing the data using Matplotlib.pyplot and Seaborn to improve model accuracy. By training multiple machine learning models—including Logistic Regression, SVM, Random Forest, and LightGBM—on historical IPL match data, I achieved an 85% accuracy in match outcome predictions. The system provides real-time analysis and strategic insights based on 8 critical input parameters.

Flipkart Review Sentiment Analysis

For this project, I conducted sentiment analysis on over 10,000 Flipkart reviews. I preprocessed the data using Pandas and NLTK, applying stopword removal, stemming, and text normalization. Using NLTK's SentimentIntensity Analyzer, I computed positive, negative, and neutral scores to derive actionable customer insights. To effectively help businesses understand customer sentiments, I created two key data visualizations: a pie chart and a word cloud, using Matplotlib.pyplot, Seaborn, and wordcloud.
